# Alpear
Mini framework PHP

## Para arrancar

* Clonar el repositorio
* 
poner Apache o Nginx como root /public


### ¿Qué te hace falta?

Apache/Nginx, PHP7 o superiores, MySQL y Composer


### Instalación

* Clonar repositorio
* Tener Composer y actualizar

```
composer update
```

* Meter credenciales de la base de datos en '/config/config.php'
